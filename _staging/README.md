# _staging

This folder holds **non-canonical** Greenhouse artifacts (drafts, snapshots, exports) that may be useful, but should not be treated as the repo source of truth.

## Rules of the road
- **Canonical repo lives at:** `~/.OpenClaw/workspace/greenhouse/`
- If a diary update exists in both staging and repo, **repo wins**.
- When new entries are generated locally, they can be staged here first, then appended into `wildling-01/DIARY.md` (with duplicate-check) and pushed.

## Current contents
- `wildling-01/DIARY.md`: local staging copy (historical)
- `greenhouse_remote_diary.md`: older remote snapshot/export
